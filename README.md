# Student Performance Dashboard (Power BI)

This project presents an interactive dashboard built using Power BI to analyze student test performance and academic results.

## Key Insights

• Student performance analysis by class
• Subject-wise test results and comparison
• Number of students per class
• Overall test performance trends

## Tools Used

• Power BI
• Data Cleaning
• Data Visualization
